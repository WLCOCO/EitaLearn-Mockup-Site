🖥️ How to Run This App

EitaLearn is a Flask (Python) web application. GitHub Pages only hosts static files, so to run the full interactive app locally:

  1. Clone the repo: git clone https://github.com/WLCOCO/EitaLearn-Mockup-Site.git
  2. Install Flask: pip install flask
  3. Run the server: python index.py
  4. Open http://127.0.0.1:5000 in your browser
